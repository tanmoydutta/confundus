# Confundus, an LIC Super Capacitor Trinket
Confundus powers an MCU driven LED matrix by means of using an LIC Super Capacitor as it's power source.

![alt text](https://github.com/tanmoydutta/confundus/blob/main/images/Confundus.png?raw=true)

It also has a matching 3D printed box to house the circuit as a keychain hanger or as an ornament that hangs next to an ID card

![alt text](https://github.com/tanmoydutta/confundus/blob/main/images/ConfundusBox.png?raw=true)
